# ☕ Automatic Coffee Powder Filling System

## 📌 Overview

The **Automatic Coffee Powder Filling System** is an academic project developed to automate the coffee powder filling process using an Arduino-based control system. The project focuses on improving filling efficiency and consistency by integrating embedded system technology into a simple automation application.

---

## 🎯 Project Objectives

- Design an automatic coffee powder filling system.
- Develop an Arduino-based control program.
- Improve the efficiency and accuracy of the filling process.
- Apply embedded system concepts in industrial automation.

---

## 👨‍💻 My Responsibilities

- Designed the 2D prototype using Adobe Illustrator.
- Developed the Arduino control program.
- Integrated hardware components for the automation system.
- Conducted system testing and troubleshooting.
- Prepared the project documentation and technical report.
